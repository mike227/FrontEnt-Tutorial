JavaScript Loops
Loops are handy, if you want to run the same code over and over again, each time with a different value.
Instead of writing:
text += cars[0] + "
";
text += cars[1] + "
";
text += cars[2] + "
";
text += cars[3] + "
";
text += cars[4] + "
";
text += cars[5] + "
";
You can write:
for (let i = 0; i < 5; i++) { text += "The number is " + i + "
"; }
